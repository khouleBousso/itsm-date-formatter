Itsm Date Formatter
=========

A small library that format date

## Installation

  `npm install @khoulzo/itsm-date-formatter`

## Usage

    var numFormatter = require('@khoulzo/itsm-date-formatter');

    var formattedNum = dateFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
